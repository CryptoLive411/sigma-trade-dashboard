# Base Sniper Web (Next.js)

Modernized with Tailwind CSS, Heroicons, and react-hot-toast.

- Tailwind config: `tailwind.config.js`
- Global CSS: `tailwind.css` (Tailwind layers) + `styles.css` (custom tokens/components)
- Toasts: `react-hot-toast` via `<Toaster />` in `_app.js`

Dev
- npm run dev
- npm run build && npm start
